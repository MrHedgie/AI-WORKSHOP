Model finansowy

Jak liczymy ROI?

Roczna produkcja = nasłonecznienie × pow. panelu × sprawność × liczba paneli × 365
Przychód roczny = produkcja [kWh] × cena energii [$/kWh]
CapEx = (cena panelu × liczba) × 1,30 + koszt działki
Zysk roczny = Przychód − OpEx roczny
ROI = Zysk roczny / CapEx
Okres zwrotu = CapEx / Zysk roczny
✅ Phoenix AZ: ROI ≈ 27,9%  |  Zwrot ≈ 3,6 lat

Architektura

Struktura projektu

  • index.html Szkielet HTML — sekcje, kontenery, kolejność skryptów
  • css/styles.css Style wspólne + bloki dla Dev A i Dev B
  • js/data.js Dane mockowe 12 lokalizacji → window.AppData
  • js/calculator.js Czyste funkcje finansowe → window.Calculator
  • js/rankings.js UI rankingów + panel kalkulatora → initRankings()
  • js/map.js Mapa SVG USA, kolor stanów wg ROI → initMap()
  • js/detail.js Widok szczegółowy, wykresy, porównanie → initDetail()
  • js/app.js Inicjalizacja — wywołuje wszystkie moduły
